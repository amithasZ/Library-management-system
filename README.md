# Simple Library Management System

A web-based library management system built with Python (Flask) and JavaScript/CSS.

## Features

- Admin can add books (title and author)
- Users can view available books
- Users can issue books
- Users can return issued books

## Installation

1. Install dependencies:
   pip install -r requirements.txt

2. Run the application:
   python app.py

3. Open your browser to http://127.0.0.1:5000/

## Usage

- Use the "Admin: Add Book" form to add new books.
- Use the "User: Issue Book" form to issue a book (enter your name and select a book).
- View issued books in the "Issued Books" section and click "Return" to return a book.
